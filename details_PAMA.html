PAMA

Personal AI Meeting Assistant

Online conference popularity has increased significantly after the COVID-19 pandemic. However, poor meeting efficiency remains a common issue. This study conducted a survey targeting individuals aged 17-30. Among 188 valid questionnaires, over 80% respondents are willing to use conference assisting software to improve meeting efficiency. To tackle this phenomenon, this project proposes developing conference assisting software incorporating natural language processing and deep learning techniques.

This system can analyze real-time dialogue during meetings, generate keywords to facilitate discussion. Meanwhile, by retrieving relevant literature records based on identified keywords, it can provide inspiration to enhance the diversity and innovativeness of discussion content, hence improving overall conference efficiency and quality. For development methodology, the Scrum agile framework will be adopted to shorten the development cycle.

This system is expected to not only enhance conference quality, but also increase user engagement and creativity, generating a virtuous cycle and increasing commercialization value.

  • Use Flask as the framework and Flask-SocketIO to establish a conference room connection.
  • Adjust Microsoft Azure model parameters to customize a dedicated speech-to-text model, and use OpenAI GPT-3.5 Turbo for natural language processing and text generation.
  • Apply Gensim/Jieba to extract keyword information from the meeting content.
  • Use D3.js to visualize keywords and create dynamic, interactive data infographics.
