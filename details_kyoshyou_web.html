Kyoshyou Blog

Travel Blog

Our travel website is designed to allow people from all over the world to share their travel experiences, which is why it supports multiple languages. There’s a “Post” section where users can view the latest shared stories, a discussion forum for talking about the content they’ve read, and a photo gallery where users can explore favorite images and link back to the posts they belong to.

The concept behind “今日醬油” (Today’s Soy Sauce) reflects our belief that by sharing travel experiences, everyone can add a bit of flavor to their daily lives—just like soy sauce enhances a meal.

Design Reference:

  • The “Post” layout takes inspiration from Instagram, with the heart icon replaced by the word すき (Japanese for “like”), and a slightly adjusted comment section.
  • The discussion forum is modeled after Dcard’s layout, incorporating the same “like” function from the post section.
  • The photo gallery is inspired by Pinterest, using a staggered, masonry-style layout that adapts to each photo’s dimensions for a visually balanced presentation.

  • Develop the UI/UX with HTML, CSS, JavaScript
  • Enhanced RWD to ensure the website is compatible with various devices.
